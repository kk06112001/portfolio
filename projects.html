Recipe Recommendation System (by Ingredients)

A content-based recipe recommendation web app built using Flask and Scikit-learn. It suggests similar recipes based on ingredient similarity using TF-IDF and cosine similarity.

Key Features:

  • Recommend recipes similar to a selected one, purely based on ingredients.
  • Filter by cuisine, course, or diet to refine results.
  • Uses TF-IDF vectorization on recipe ingredients for computing similarity.
  • Lightweight and easy to use via a clean Flask-powered UI.

How It Works:

  • Recipes are vectorized using TfidfVectorizer based on their ingredient lists.
  • When a user selects a cuisine, course, or diet, the app finds matching recipes.
  • For the first filtered recipe, it recommends the top 8 similar recipes based on cosine similarity.
